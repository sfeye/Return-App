# Return-App

Return App - Consumer

1. create branch off of master
2. checkout new branch
3. make code changes, commit frequently (view changes -> stage all -> commit -> push)
4. open pull request to main
5. wait for review from team
6. squash & merge pull request into master
7. Delete branch locally
